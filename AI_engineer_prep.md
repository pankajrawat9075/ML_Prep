## 1. Introduction
- What is an AI Engineer?
  - An AI Engineer is a software professional who designs, builds, and deploys artificial intelligence systems and applications. They work with machine learning models, large language models (LLMs), and AI frameworks to create intelligent solutions that can learn, reason, and automate tasks. AI Engineers bridge the gap between data science research and production-ready AI applications.

## 2. Core Concepts
- Data Structures & Algorithms
- Machine Learning & Deep Learning
- GenAI & LLMs
- System Design
- Prompt Engineering

### Key skills required (not everything required)
- **Programming Languages**: Python (primary), JavaScript/TypeScript
- **LLM Frameworks**: LangChain, LangGraph, LlamaIndex
- **AI/ML Concepts**: Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), Prompt Engineering, Fine-tuning
- **Model Context Protocol (MCP)**: Building MCP servers and integrating tools
- **Open-Source Frameworks**: Hugging Face Transformers, Ollama, vLLM, FastAPI
- **Data Structures & Algorithms (DSA)**: Problem-solving and optimization
- **Databases**: SQL (PostgreSQL, MySQL), NoSQL (MongoDB, Redis), Vector Databases (Pinecone, Weaviate, ChromaDB, Qdrant)
- **Cloud & DevOps**: AWS/GCP/Azure, Docker, Kubernetes, CI/CD pipelines
- **Version Control**: Git, GitHub
- **Soft Skills**: System design, communication, problem-solving
