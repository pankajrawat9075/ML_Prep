
# Teradata AI Engineer - Interview Experience

## OA / Screening Round
- **Format:** Online Assessment
- **Duration:** 2 hrs
- **Topics Covered:**
  - Data Structures & Algorithms / Coding problems
    - Medium level sliding window
    - Heap + Linked list based problem
  - Machine Learning Concepts
    - Design AI agent from scratch (AI help allowed)
    - RAG, embeddings creation, Agent orchestration
- **Outcome:** Selected


## Round 1
- **Format:** Technical (AI / GenAI) Interview
- **Duration:** 1 hrs
- **Topics Covered:**
  - Core GenAI Concepts
    - Hallucinations & mitigation strategies
    - Context window management
    - Prompt engineering & optimization
    - How llms deal with streaming data, pdf files
  - LangGraph Single Agent Implementation
    - Nodes, conditional edges, state management
    - Pseudo code allowed
  - AI System Design (past projects)
    - Multi agent orchestration, custom tools
    - challenges and solutions proposed
- **Outcome:** Selected


## Round 2
- **Format:** AI System Design Interview
- **Duration:** 2 hrs
- **Topics Covered:**
  - Internal Knowledge Base Agent
    - Confluence, presentations, documents, and emails integration
    - Chat interface implementation
  - Multi-Agent System Design
    - Coordination mechanisms across teams within an organization
  - Project Deep Dive
    - Past project design, challenges encountered, and solutions proposed
- **Outcome:** Selected


## Round 3
- **Format:** Managerial Round
- **Duration:** 1 hr
- **Topics Covered:**
  - Career Goals & Motivation
    - Long-term vision in AI/ML
    - Why Teradata
  - Team Collaboration & Conflict Resolution
    - How you handle disagreements with seniors
    - Approach to differing technical opinions
  - Work-Life Balance & Deadlines
    - Managing extended work hours during critical deadlines
    - Prioritization strategies
  - Feedback & Growth Mindset
    - How you receive and implement feedback
    - Examples of past improvements based on feedback
  - Company Fit & Culture Questions
    - Alignment with organizational values
    - Team integration expectations
- **Outcome:** Selected


## Overall Experience

- **Difficulty Level:** Medium
- **Key Learnings:** Technical depth is essential, along with a focus on system design
- **Tips & Advice:**
        - Develop systems-level thinking
        - Prepare past projects thoroughly
        - Practice medium-level DSA problems
        - Approach it as a collaboration rather than a traditional interview
